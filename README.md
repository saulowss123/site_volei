Arena Copacabana - Sistema de Reservas de Quadras
Este projeto é um sistema web para gerenciamento e reserva de quadras esportivas na Arena Copacabana, em São José do Rio Preto. Ele permite que usuários realizem reservas de quadras de vôlei de praia, beach tênis, futevôlei e espaço para churrasco, além de gerenciar seus times e planos. O sistema também possui um painel administrativo para controle de reservas, quadras, modalidades, preços e usuários.

Funcionalidades
Página Principal: Apresenta informações sobre a Arena, serviços oferecidos, quadras disponíveis, horários de funcionamento e contato via WhatsApp.
Reserva de Quadras: Usuários podem selecionar quadra, dia e horário, e realizar reservas após login.
Gestão de Usuário: Área do usuário para visualizar reservas, criar times, editar perfil e gerenciar plano.
Administração: Painel para administradores gerencia reservas, quadras, modalidades, preços e usuários.
Responsividade: Layout adaptado para dispositivos móveis.
Como funciona
O usuário navega pela página principal (index.html), visualiza os serviços e quadras disponíveis e pode iniciar uma reserva.
Para reservar, é necessário fazer login ou criar uma conta (login.html).
Após login, o usuário acessa sua área (usuario.html), onde pode acompanhar reservas, criar times e editar seu perfil.
O painel administrativo (admin.html) permite ao gestor aprovar ou rejeitar reservas, alterar preços, ativar/desativar quadras e gerenciar modalidades e usuários.
O sistema utiliza arquivos JavaScript para manipulação dinâmica dos dados e interface, e arquivos CSS para estilização.
